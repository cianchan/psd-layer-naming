🎨 PSD 图层自动重命名工具

批量处理 PSD 文件：自动删除隐藏图层、解散图层组、应用蒙版、统一重命名图层

🍎 macOS 专属
Adobe Photoshop 2023+
Python 3.9+
本地运行 数据不外传
功能一览
🗑️

删除隐藏图层

自动识别并删除所有不可见图层，保持文件整洁

📂

解散图层组

将所有 Group 组拆解为独立图层，便于后续使用

🎭

应用图层蒙版

自动 Apply 所有像素蒙版，消除蒙版链接

✏️

统一重命名

按图层类型智能命名，输出规范的图层结构

🔓

解锁所有图层

自动解除图层锁定，避免处理时报错

📦

批量处理

一次选择整个文件夹，批量处理所有 PSD

命名规则
图层类型命名结果说明
文本图层 text  text1 text2… 单个文本层命名为 text；多个时按顺序编号
像素 / 智能对象 scenebg 统一命名 scenebg，不编号
形状 / 填充图层 stickerbg  stickerbg1… 多个时按顺序编号
边框图层（四边环绕） frame 自动识别覆盖画布 ≥80% 的形状图层
无文本图层的 PSD 跳过 不含文本图层的文件直接跳过，不输出
运行要求
  • macOS 12 Monterey 或更高版本
  • Adobe Photoshop 2023 / 2024 / 2025（已激活）
  • Python 3.9 或更高版本
  • pip 包：flask、python-docx
安装步骤
1

克隆仓库

打开终端，运行以下命令（需要 GitHub 账号且已被加入仓库）

git clone https://github.com/cianchan/psd-layer-naming.git
cd psd-layer-naming
2

安装依赖

pip3 install flask python-docx
3

启动工具

python3 app.py

启动后在浏览器访问 http://127.0.0.1:7861

4

设置开机自启（可选）

运行下方命令，让工具在每次登录 Mac 时自动在后台启动

curl -fsSL https://raw.githubusercontent.com/cianchan/psd-layer-naming/main/install-service.sh | bash
